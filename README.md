#  Movie App

##  Project Idea

Movie App is a modern web application that allows users to discover trending movies, search for movies, view detailed information about each movie, and manage their favorite movies using TMDB API.

---

##  Objectives

- Discover trending movies.
- Search for movies easily.
- View movie details.
- Practice React.js concepts.
- Build a responsive and user-friendly interface.

---

##  Technologies Used

- React.js
- React Router DOM
- Bootstrap
- Axios
- TMDB API
- HTML5
- CSS3
- JavaScript (ES6)

---

## Main Features

- Browse trending movies.
- Search for movies.
- View movie details.
- Add movies to favorites.
- Responsive design for all devices.
- Fast navigation using React Router.

##  How to Run the Project

### Clone the repository

```bash
git clone https://github.com/ItcProjects-R4/SHR4_SWD2_S1_PROJECT5.git
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm start
```

### Open in browser

```bash
http://localhost:3000
```
